# Kumpulan Mini Proyek Go-lang

Disini tempat ku menyimpan hasil praktik ketika belajar bahasa (Go).
Jika penasaran bisa langsung ambil aja

```bash
git clone https://github.com/masputrawae/go-mini-projects.git
```

## Struktur

Struktur proyek campur, mengingat ini cuma penyimpanan aja,
dan bisa berubah kapanpun.

```bash
.
├── level-01
│   ├── go.mod
│   └── kalkulator
│       └── main.go
├── LICENSE
└── README.md
```
